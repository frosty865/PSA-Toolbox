# Infrastructure Dependency Assessment report template

DOCX export **must** use the single canonical template at:

**`ADA/report template.docx`** (this folder, file name: `report template.docx`)

- Place the narrative-only anchor template here (with `[[SNAPSHOT_*]]` and `[[INFRA_*]]` tokens).
- Do **not** use `Asset Dependency Assessment Report_BLANK.docx` or any "formatted output" doc as the export template.
- If you have `ADA_report_template_NARRATIVE_ONLY.docx`, copy it here as **`report template.docx`**.

Export will fail with "Wrong template: export must use /ADA/report template.docx" if any other path is used.

**Template maintenance (manual):** On the cover page, remove any duplicate lines (e.g. "Infrastructure Dependency Assessment" or "TechSecPros platform" appearing more than once).
