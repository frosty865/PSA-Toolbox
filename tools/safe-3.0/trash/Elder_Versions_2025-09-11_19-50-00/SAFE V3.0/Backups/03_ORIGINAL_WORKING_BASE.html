Welcome to SAFE V3.0

Comprehensive Security Assessment for Facilities and Events

Assessment Purpose: This tool helps security professionals conduct comprehensive facility security assessments using the Protective Security Advisor (PSA) methodology. The assessment identifies vulnerabilities and provides specific options for consideration to enhance security posture.
Important: This assessment is designed to facilitate discussion and identify areas for improvement. There are no "right" or "wrong" answers - we're here to understand your current approach and explore opportunities for enhancement.

Getting Started

To begin your security assessment:

  1. Complete the Facility Information section with basic details about your facility
  2. Work through each section systematically, answering questions about your current security practices
  3. Review the Report Review section to see auto-generated findings
  4. Generate your Final Report for executive presentation
