Security Assessment for First Entry - Comprehensive Security Assessment Tool

What is SAFE V3.0?

SAFE V3.0 stands for "The Security Assessment for First Entry (SAFE) V3.0." It is a comprehensive security assessment tool developed by the Cybersecurity and Infrastructure Security Agency (CISA) to help organizations evaluate and improve their physical security posture.

The assessment examines multiple security domains, including:

  • Information Sharing - Communication protocols and intelligence sharing
  • Security Plans - Emergency and security planning documents
  • Physical Security - Access control and perimeter security
  • Security Systems - Surveillance and monitoring systems

Simple Terms

SAFE V3.0 identifies the path of least resistance into your facility - essentially finding the easiest ways someone could gain unauthorized access. It also evaluates your facility's ability to detect, respond to, and manage security incidents when they occur.

Vulnerability Assessment

Finding security gaps that could be exploited

Response Capability

Evaluating incident detection and response readiness

Assessment Methodology

The assessment evaluates your facility against established security standards and best practices. Each question within the assessment is designed to measure compliance with specific security requirements.

Responses to questions are categorized into three types, each with a clear meaning:

YES

Your facility meets this security standard.

NO

Your facility does not meet this security standard. This is identified as a vulnerability.

N/A

This standard does not apply to your facility.

Getting Started

To begin your security assessment:

  1. Complete the Facility Information section with basic details about your facility
  2. Work through each section systematically, answering questions about your current security practices
  3. Review the Report Review section to see auto-generated findings
  4. Generate your Executive Summary and Final Report for presentation
Auto-Save Feature: Your progress is automatically saved as you work. Use the "Restore Auto-Save" button above if you need to recover your previous session.
