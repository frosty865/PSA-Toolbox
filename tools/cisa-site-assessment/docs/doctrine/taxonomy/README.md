# Taxonomy Directory (Read-Only Reference)

## Purpose

This directory contains **read-only references** to taxonomy that is authoritative in `psa_engine`.

## What Belongs Here

- Taxonomy snapshot exports (if any)
- Taxonomy reference files for UI display
- **NOT allowed**: Modifications to taxonomy, new taxonomy definitions

## Authority

**psa-rebuild is NOT authoritative for taxonomy.**

All taxonomy is authoritative in `psa_engine`. Files here are read-only references only.

## Classification

- **Type**: Read-only reference
- **Authority**: `psa_engine`
- **Modifiable**: No
- **Purpose**: Reference for UI display only
