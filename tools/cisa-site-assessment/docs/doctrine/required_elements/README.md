# Required Elements Directory (Read-Only Reference)

## Purpose

This directory contains **read-only references** to required element sets that are authoritative in `psa_engine`.

## What Belongs Here

- Snapshot files of required element sets from `psa_engine`
- Reference copies for UI display purposes
- **NOT allowed**: New required elements, modifications to existing elements

## Authority

**psa-rebuild is NOT authoritative for required elements.**

All required elements are authoritative in `psa_engine`. Files here are read-only references only.

## Classification

- **Type**: Read-only reference
- **Authority**: `psa_engine`
- **Modifiable**: No
- **Purpose**: Reference for UI display only
