# Doctrine Rules Directory (Read-Only Reference)

## Purpose

This directory contains **read-only references** to doctrine rules that are authoritative in `psa_engine`.

## What Belongs Here

- Rule documentation for UI display context
- References to scoring rules, N/A gating rules, etc.
- **NOT allowed**: New rules, modifications to existing rules

## Authority

**psa-rebuild is NOT authoritative for doctrine rules.**

All doctrine rules are authoritative in `psa_engine`. Files here are read-only references only.

## Classification

- **Type**: Read-only reference
- **Authority**: `psa_engine`
- **Modifiable**: No
- **Purpose**: Reference for UI display context only
