# Decisions Directory

## Purpose

This directory contains **rationale, intent, and tradeoffs** for architectural and design decisions made in psa-rebuild.

## What Belongs Here

- "Why we chose X" documents
- "Why we deferred Y" explanations
- Architectural tradeoffs
- Design rationale
- Historical context for decisions
- UI/UX design decisions

## What Does NOT Belong Here

- **Doctrine content**: Doctrine belongs in `psa_engine` or `docs/doctrine/` (read-only)
- **Process documentation**: Process docs belong in `docs/process/`
- **Generated artifacts**: Artifacts belong in `docs/artifacts/`
- **General documentation**: General docs belong in `docs/`

## Classification

- **Type**: Human-readable rationale
- **Authority**: Historical context only
- **Modifiable**: Yes (to add new decisions)
- **Purpose**: Explain why decisions were made

## Examples

- "Why we chose Next.js"
- "Tradeoffs in component architecture"
- "Decision to defer feature X"
- UI/UX design decisions
