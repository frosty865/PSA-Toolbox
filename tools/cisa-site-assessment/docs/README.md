# Documentation Directory

## Purpose

This directory contains **general documentation, onboarding guides, and project-wide documentation** for psa-rebuild.

## What Belongs Here

- Developer guides
- Onboarding documentation
- Setup instructions
- API documentation
- General project documentation
- Cross-project coordination docs

## What Does NOT Belong Here

- **Doctrine content**: Doctrine belongs in `psa_engine` or `docs/doctrine/` (read-only)
- **Process documentation**: Process-specific docs belong in `docs/process/`
- **Decisions**: Decision rationale belongs in `docs/decisions/`
- **Generated artifacts**: Artifacts belong in `docs/artifacts/`

## Classification

- **Type**: General documentation
- **Authority**: psa-rebuild (for psa-rebuild-specific content)
- **Modifiable**: Yes
- **Purpose**: Help developers understand and use psa-rebuild

## Key Documents

- `AUTHORITY.md`: Project authority boundaries
- Setup and configuration guides
- API and component documentation
