## Archived Deprecated APIs

### 1. `/api/assessment/scoring` ✅ ARCHIVED

- **Original Path:** `psa_rebuild/app/api/assessment/scoring/route.ts`
- **Archive Path:** `archive/apis/deprecated/assessment-scoring/`
- **Status:** ⚠️ Still in use but deprecated
- **Reason:** Legacy route using `documentId` parameter; conflicts with assessment-scoped architecture
- **Replacement:** `/api/runtime/assessments/[assessmentId]/results`
- **Still Referenced By:**
  - `app/assessments/[assessmentId]/results/page.tsx` (line 8)
  - `app/api/runtime/assessments/[assessmentId]/results/route.ts` (comments)

**Migration Required:**
- Update `app/assessments/[assessmentId]/results/page.tsx` to use the new assessment-scoped endpoint
- Remove deprecated endpoint after migration is complete

### 2. `/api/admin/module-drafts/[draftId]/scenario-context` ✅ ARCHIVED

- **Original Path:** `psa_rebuild/app/api/admin/module-drafts/[draftId]/scenario-context/route.ts`
- **Archive Path:** `archive/apis/deprecated/module-drafts-scenario-context/`
- **Status:** No-op, can be removed
- **Reason:** Scenario context is no longer used (template-first doctrine). Endpoint is already a no-op.
- **Replacement:** No replacement needed. Scenario context is no longer part of module generation.

**Action Required:**
- Remove any references to this endpoint
- Can be safely removed from the codebase
